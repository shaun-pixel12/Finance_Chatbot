# AGENTS.md

## 프로젝트 소개
LangChain을 활용한 챗봇 에이전트 프로젝트.
LLM이 Yahoo Finance에 공시된 한국, 미국의 증시 및 주가 정보를 끌어와서 요약을 볼 수 있게 해주는 에이전트.

## 환경
- Python 3.11
- 패키지 관리: `pip-chill > requirements.txt`
- 가상환경: conda (`langchain-env`)

## 코딩 스타일
- 코드보다 설명 위주로, 비개발자도 읽을 수 있도록 주석을 충분히 달기
- 함수나 변수 이름은 역할이 바로 드러나도록 한국어 주석으로 보완하기
- 한 파일에 너무 많은 기능을 넣지 말고, 기능별로 파일을 분리하기

## 반드시 지킬 것 (Do)
- LangChain 코드를 작성할 때는 반드시 아래 공식 문서를 먼저 확인할 것
  👉 https://docs.langchain.com/oss/python/langchain/overview.md
- 코드를 변경하기 전에 먼저 어떤 방식으로 수정할지 설명하고 확인받기
- `.env` 파일에 API 키를 저장하고, 절대 코드에 직접 적지 않기

## 하지 말 것 (Don't)
- 공식 문서에 없는 함수나 방법을 임의로 만들어 쓰지 않기
- 한 번에 너무 많은 코드를 수정하지 않기 (작은 단위로 나눠서 수정)
- `.env` 파일을 GitHub에 올리지 않기
